# Glance Markdown Editor

Glance is a portable desktop application for editing Markdown files with a live preview.

## Features

- Open folders containing Markdown files
- Browse files in a tree view
- Edit Markdown files in tabbed interface
- Live preview of Markdown content
- Support for GitHub Flavored Markdown

## Getting Started

1. Open a folder containing Markdown files using "File > Open Folder"
2. Browse files in the left panel
3. Click on a Markdown file to open it in the editor
4. Edit the Markdown content
5. See live preview updates in the right panel

## Keyboard Shortcuts

- Ctrl+O: Open Folder
- Ctrl+Q: Quit
- F1: About

## Supported File Types

- .md (Markdown)
- .markdown
- .mdown
- .mkd
